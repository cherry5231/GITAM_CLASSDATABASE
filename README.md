# GITAM_CLASSDATABASE
# GITAM University Info System

A Python-based Object-Oriented Programming project that organizes and displays structured information about GITAM University using class-based design.

---

##  Project Overview

**GITACLASSDATABASE** is a structured Python project that models real-world university information using Object-Oriented Programming.

It includes:

-  Campus information (Visakhapatnam, Hyderabad, Bangalore)
-  Schools and branches
-  Clubs and student communities
-  Fests and cultural events
-  Library system
-  Hostel facilities
-  Sports activities
-  Placements
-  Student achievements
- Campus life details

All data is displayed in a clean, structured format using Python classes and methods.

---

##  Purpose of This Project

The main purpose of this project is to **write, organize, and display data using only class-based functions (Object-Oriented Programming methods)**.

It focuses on:
- Using classes as data containers
- Using methods to display structured information
- Avoiding procedural programming
- Practicing real-world data modeling using Python OOP

---

##  Output Feature

- The program generates a **`.txt` file (`gitam_info.txt`)**
- The file contains all structured output data
- It is fully readable and can be opened using **Notepad or any text editor**
- It works as a simple database-like text export system

---

##  Concepts Used

- Classes & Objects
- Constructors (`__init__`)
- Methods
- Lists & Iteration
- Encapsulation (basic OOP concept)
- File handling (`sys.stdout` redirection)
- Structured data representation
